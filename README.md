## 介绍

第一门课程是Berkeley的基于Python语言的SICP（计算机程序构造与解释）。该课程是Berkeley大一新生的第一门计算机课程，CS61A介绍了编程和计算机科学，重点是**抽象技术**作为管理程序复杂性的手段。

技术包括

-   过程抽象；
-   使用递归、高阶函数、生成器和流控制抽象；
-   使用接口、对象、类和泛型操作符进行数据抽象；
-   使用解释器和宏进行语言抽象。
-   提供编程范例，包括函数式、面向对象和声明式方法。
